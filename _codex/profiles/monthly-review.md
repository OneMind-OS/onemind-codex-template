# Monthly Review Profile

**When**: 1st of each month, ~60 minutes
**Goal**: See the bigger picture, course-correct heading into the new month

## Checklist

- [ ] Read all 4 weekly reviews from last month
- [ ] Fill in monthly `_codex/templates/review.md` → save to `HP/29 Life Systems/Reviews/`
- [ ] Complete the Quadrant Check (one paragraph per quadrant: UI / HP / LE / GE)
- [ ] Review personal finance metrics (see `HP/30 Finance/`)
- [ ] Review health metrics (see `HP/27 Body/`)
- [ ] Archive completed projects, update status on stale ones
- [ ] Update `_codex/system/active-goals.md` — reprioritize if needed
- [ ] Run vault maintenance checklist (see `_codex/skills/maintain.md`)

## The Key Question

*"If this month repeated for the next 12 months, where would I end up?"*
If the answer aligns with your goals — keep going.
If not — change something.
